TeamBotsTM Users

TeamBots is being used by a number of researchers and educators around the world. If you'd like to be added to this list, please send e-mail.

  • In research:

    • Mitchell Potter and Lisa Meeden at the Naval Research Laboratory are using JavaSoccer as a domain for the investigation of evolutionary computation. They plan to run the evolved strategies on Nomad Scout robots.

    • Ashwin Ram and Irfan Essa at Georgia Tech are leading a project aimed at developing a "robot pet" called Pepe. The long-term objective of this research is to build an intelligent, adaptive, user-friendly agent that displays different emotional states and a wide range of behaviors. Their software controls an ISR Pebbles robot using TeamBots.

    • Paul Rybski and Sascha A. Stoeter at the University of Minnesota are using TeamBots as a platform for the investigation of robot navigation across variable terrain.

    • Hiroaki Kitano at Sony CSL is using JavaSoccer as a platform for the development of robot soccer strategies for RoboCup Competition.

    • Darrin Bentivegna at Georgia Tech's Mobile Robot Laboratory is using TeamBots as a platform to investigate how groups of robots can be commanded by a human using visual gestures/signals. He has integrated TeamBots with one of Newton Lab's Cognachrome vision system on a Nomad 150 robot.

    • TeamBots software controlled Georgia Tech's winning multi-robot entry in the 1997 AAAI Mobile Robot Competition.

    • Asif Razaq at Queen Mary University of London is implementing support for Khepera robots in TeamBots. He is investigating robot soccer using the Khepera platform.

  • In education:

    • CS 4324 ``Intelligent Robotics and Computer Vision'' taught at Georgia Tech by Chris Atkeson; JavaSoccer used as a class project. Several student-developed teams from this class are included in the TeamBots distribution.

    • CS 7100 ``Introduction to Graduate Studies'' taught at Georgia Tech by Irfan Essa; JavaSoccer used as a class project. Several student-developed teams from this class are included in the TeamBots distribution.

    • 2D5350 ``Behavior-based Robotics'' taught at KTH (Sweden) by Ronald Arkin; TeamBots used as a platform for class projects.

    • CS 260 ``Java'' taught at the University of North Dakota by Henry Hexmoor; JavaSoccer suggested as a class project.

    • CS 599 ``Mobile Robots and Multi-Robot Systems'' taught at the University of Southern California by Maja Mataric; TeamBots suggested as a platform for class projects.
